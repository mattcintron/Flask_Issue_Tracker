Welcome To The Flask Issue Tracker

Flask Issue Tracker is a web application built using Python, Javascript, bootstrap and SQL databases in the flask framework. The site allows for users to track work on certain projects and open or close issues on those projects users have Project pages and can upload images to them witch are tracked and re-sized for space on the server with the python lib Pillows all data entry's are saved and searchable through the sites link to a SQL data base passwords can be reset via email
