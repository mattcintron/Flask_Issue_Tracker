Flask Issue Tracker:

A web application built using Python, Javascript, bootstrap and SQL databases. The site allows for users 
to track work on certain projects and open or close issues on those projects users have profile pages 
and can upload images to them witch are tracked and resized for space on the server with the pyhton lib 
pillows all data entrys are saved and searchable through the sites link to a SQL data base paswords 
can be reset via email.

steps to run :

1. Build a virtual enviroment in python 3.6 or higher and move in code

2. Install libs using the requiremnets.txt file 

3. Run the run.py file
